# rubik bundle 

From [here](https://ctan.org/pkg/rubik)

## Packages
- [rubikcube](https://2.mirrors.in.sahilister.net/ctan/macros/latex/contrib/rubik/rubikcube.pdf) provides commands for typesetting Rubik cubes and their transformations,
- [rubiktwocube](https://mirrors.sukhala.in/ctan/macros/latex/contrib/rubik/rubikpatterns.pdf) provides commands for typesetting Rubik twocubes and their transformations,
- [rubikrotation](https://mirrors.sukhala.in/ctan/macros/latex/contrib/rubik/rubikrotation.pdf) which can process a sequence of Rubik rotation moves, with the help of a Perl package executed via \write18 (shell escape) commands, and
- [rubikpatterns](https://mirrors.sukhala.in/ctan/macros/latex/contrib/rubik/rubikrotation.pdf), a collection of well known patterns and their associated rotation sequences.
